# Rust Ollama Demo

## Description

This is a command-line interface (CLI) chat application built in Rust that integrates with Ollama to provide AI-powered conversations. It uses the "ministral-3:8b" model for generating responses and maintains a conversation history.

## Prerequisites

- Rust programming language installed. You can install it from [rust-lang.org](https://rust-lang.org/tools/install/).
- Ollama running in the background. Download and install from [ollama.com](https://ollama.com/) and ensure the "ministral-3:8b" model is available.

## Running the Code

To run the application, navigate to the project directory and use the following command:

```bash
cargo run
```

Once running, you can start chatting with the AI by typing your messages. Type 'exit' to quit the application.
